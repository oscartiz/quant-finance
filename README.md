# 📈 Quant Finance

A personal learning repository for quantitative finance — covering mathematical modeling, statistical methods, derivatives pricing, and algorithmic strategy research.

---

## 📂 Repository Structure

```
quant-finance/
├── 01_math_foundations/       # Linear algebra, calculus, probability, stochastic processes
├── 02_statistics/             # Regression, time series, hypothesis testing, MLE
├── 03_derivatives_pricing/    # Black-Scholes, binomial trees, Monte Carlo, Greeks
├── 04_portfolio_theory/       # MPT, CAPM, factor models, risk parity
├── 05_fixed_income/           # Bond pricing, duration, yield curves, swaps
├── 06_risk_management/        # VaR, CVaR, stress testing, drawdown analysis
├── 07_alpha_research/         # Factor discovery, signal construction, backtesting
├── 08_market_microstructure/  # Order book dynamics, execution, bid-ask spread
├── 09_ml_in_finance/          # ML models applied to pricing, prediction, regime detection
├── notebooks/                 # Exploratory Jupyter notebooks
├── data/                      # Sample datasets (gitignored large files)
└── utils/                     # Shared Python utilities
```

## 🚀 Getting Started

```bash
# Create virtual environment
python -m venv .venv && source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Launch notebooks
jupyter lab
```
